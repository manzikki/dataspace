Dataspace management for CVS files.


https://sourceforge.net/projects/simple-dataspace-management/ niinimakim@webster.ac.th
This Python/Web based software lets you run a site for managing data sources for data integration. It supports upload of CVS files and entering/editing their meta data.
Three main use cases: 1 use the dataspace to prepare your files, 2 use the dataspace system's web GUI to merge/prepare the data and download it as CSV 3 export the data as RDF.

How to use it

Functionality available without login: Download files, Download as RDF.
Functionality available as admin: Upload files, edit file meta data, create collections, merge files and download the result.

How to deploy it

Please see the install.txt file at https://sourceforge.net/projects/simple-dataspace-management/
